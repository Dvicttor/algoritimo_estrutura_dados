# 📊 Algoritmos e Estrutura de Dados em Python

Repositório dedicado ao estudo e prática de **algoritmos e estruturas de dados**, utilizando a linguagem Python 🐍

---

## 📚 Sobre

Este projeto reúne implementações e exercícios voltados para o desenvolvimento da lógica de programação e compreensão de estruturas fundamentais da computação.

O objetivo é evoluir gradualmente, saindo do básico até conceitos mais avançados.

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/Dvicttor/algoritimo_estrutura_dados
```

2. Acesse o diretório:

```bash
cd algoritimo_estrutura_dados
```

3. Execute um arquivo Python:

```bash
python nome_do_arquivo.py
```
